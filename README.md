# 我的项目
## 1.吉林化工学院教务系统
### 应用介绍
将web应用转为原生的android应用。利用volley实现登录以及网页源代码的获取，使用jsoup解析提取网页信息，可查看新闻、公告等，以及查询个人信息、课表、成绩、考试安排等信息。
